# Exercice 3 - Boucle

## Exercice 3.1 : Table de multiplication
Permettre à l'utilisateur d'encoder une table et un nombre d'operation.
Afficher la table de multiplication le format suivant :
```html
<ul>
  <li>1 x 3 = 3</li>
  <li>2 x 3 = 6</li>
  <li>3 x 3 = 9</li>
  ...
  <li>10 x 3 = 30</li>
</ul>
```
Si l'utilisateur encode une nouvelle table, effacer le résultat précédent.
