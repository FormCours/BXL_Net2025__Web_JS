# Demo Promesse

## Object promesse
Un promesse ne peut avoir que 3 status possible : 
 - Pending
 - Fulfilled
 - Rejected

## Simulation
Pour mettre en pratique, on va créer la simulation de la préparation d'un repas

### Café
- Allumer la machine à café (1.5s)
- Faire le café (5s)
- Servir le café (0.5s)

### Salade
- Couper les 3 légumes (2s par légumes)
- Servir la salade (2s)

### Viande
- Faire chauffé la poele (3s)
- Faire cuir la viande (5s)
  → Risque de foirage : 1 chance sur 5
- Servir la viande (1s)
